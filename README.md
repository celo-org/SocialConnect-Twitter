## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

-   Nodejs
-   Yarn

### Installation

1. Clone the repo

    ```sh
    git clone https://github.com/celo-org/SocialConnect-Twitter.git
    ```

2. Install NPM packages

    ```sh
    yarn
    ```

3. Add variables in `.env` (refer to `.env.example`)

4. Start App

    ```sh
    yarn run react-app
    ```
